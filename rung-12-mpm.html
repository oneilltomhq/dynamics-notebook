01 · the handshake

Nine numbers between two worlds

Before anything moves, look at the interface. One particle floats over a ghost grid — move your pointer and it follows. At every instant the particle talks to exactly nine grid nodes, the 3×3 block around it, and each node gets a share wi·wj computed from a quadratic B-spline per axis. Watch the shares slosh smoothly from node to node as the particle crosses a cell.

// fx = particle position in cell units, minus its base node w0 = 0.5·(1.5 − fx)² // left node w1 = 0.75 − (fx − 1)² // middle node w2 = 0.5·(fx − 0.5)² // right node · w0 + w1 + w2 = 1, always

This handshake is the whole interface between the two worlds — nine numbers, and they change smoothly in every direction the particle moves. No jumps, ever: as the particle drifts right, the left node's share fades out exactly as the new right node's fades in. Rung 3's fade curve did this job for the noise lattice; these B-splines do the same job here, and the smoothness is load-bearing — a jump in the weights would be a kick of force from nowhere. Everything in the next section passes through these nine numbers, both directions.

02 · jelly, sand-free

Jelly

The real thing — classic MLS-MPM (Hu et al. 2018, the famous 88-line formulation). Each particle carries position, velocity, an affine matrix C, and a 2×2 matrix F that records how its neighborhood has deformed. Each substep: particles scatter mass, momentum and stress to the grid (P2G), the grid does one momentum step and enforces the walls, and the particles gather everything back (G2P). Drag through the blocks to poke them.

// one substep — count the trades grid.clear() // scratch paper, wiped every substep for p of particles: // P2G — trade one P = μ(F − F⁻ᵀ) + λ·log(det F)·F⁻ᵀ // stress — rung 1's spring law, matrix-sized scatter mass, momentum, −dt·vol·4/dx²·(P·Fᵀ) // through §01's nine weights for node of grid: // grid — rung 5's grid, demoted to calculator v = momentum / mass + g·dt // rung 1's integrator, once per node walls: zero v into the border // rung 5's walls, same trick for p of particles: // G2P — trade two gather v, C · x += v·dt · F ← (I + dt·C)·F // memory updated, grid forgotten

Count the trades: P2G, one grid step, G2P — then the grid is wiped. Rung 5's grid, which used to be the fluid, is demoted to a calculator here: it exists for one momentum step and remembers nothing. The memory moved into F. Squash a block and F records the squash; that record is the spring that pushes back — the stress formula reads F, compares it to identity, and pushes toward rest, exactly rung 1's law with matrices in place of numbers. Colors show det F: amber is compression, cyan is stretch. Stiffen E and the blocks snap back harder; soften it and they slump like custard.

03 · f, the memory

Where did my neighborhood go

F deserves its own toy. It answers one question for each particle: where did my neighborhood go? Below, a square of dots sits in its rest pose. The two clay handles are the columns of F — where the unit x and y arrows end up. Drag them and every dot maps through F; the dashed unit circle maps to an ellipse, and det F reports the area change.

Identity means relaxed — every dot exactly where it started, zero stored energy. Any deviation is stored elastic energy, and the stress formula in section 02 is nothing but a spring law written for matrices: it measures how far F sits from identity and pushes back. Squeeze the handles together (det < 1, amber) and the material wants to re-inflate; pull them apart (det > 1, cyan) and it wants to shrink; cross them and it has turned inside out, which is why the sim guards det F away from zero. That is the last tool in the box. Twelve rungs: a spring, a wander, fields, curl, a fluid, chaos, sync, gravity, light, growth, particle fields, and now the trade that binds particles to grids. The storm assembles all twelve into one sky.
